Element

Tag

Name

Description
<header>

Header

Header is the content at the top of the page, it may include a logo, navigational links or a search bar.
<nav>

Navigation

For encapsulates the page’s navigational links. It is often placed inside the header or footer.
<main>

Main

The main content of a page between the header and the footer areas.
<footer>

Footer

To includes the page’s footer content at the bottom of the <body>.

Figure

Tag

Name

Description
<figure>

Figure

Used to encapsulate media such as an image, diagram, or code snippet.
<figcaption>

Figcaption

Used to describe the media encapsulated within the <figure> element.

Section

Tag

Name

Description
<section>

Section

Defines elements in a document, such as chapters, headings, or any other area of the document with the same theme.
<article>

Article

Holds content that makes sense on its own such as articles, blogs, and comments.
